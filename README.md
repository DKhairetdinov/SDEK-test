# SDEK News Component

Универсальный компонент новостного блока. Компонент получает данные с 
существующего API и отображает новости с пагинацией.

## Стек технологий
- **React**
- **TypeScript**

## Функционал
- Универасальный компонент для разных типов новостей.
- Состояния загрузок и обработка ошибок.
- Погинация с кешированием данных.
- Ленивая загрузка.

## Иструкиця по запуску
1. Клонировать репозиторий
```bash
git clone https://github.com/DKhairetdinov/SDEK-test.git
```
2. Устновить зависимости:
```bash
npm install
```
3. Запустить проект:
``` bash
npm run dev
```
