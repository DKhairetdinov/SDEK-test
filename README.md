# SDEK News Component

Универсальный компонент новостного блока. Компонент получает данные с 
существующего API и отображает новости.

## Стек технологий
- **React**
- **TypeScript**
- **CSS Modules**
- **Tabler Icons**
- **Axios**

## Реализованный функционал
- **Универсальность**: Один компонент `NewsFeed` поддерживает разные режимы отображения.
- **Автономность**: Блоки работают независимо друг от друга.
- **Адаптивность**: 
  - На Desktop: горизонтальные карточки, картинка слева.
  - На Mobile: вертикальные карточки, картинка сверху.
- **UX/UI**:
  - Реализованы **Skeletons** для ожидания загрузки.
  - Обработка пустого состояния с тележкой.
  - Обработка ошибки доступа к API.
  - Интерактивные кнопки пагинации.

## Иструкция по запуску
1. Клонировать репозиторий:

`git clone https://github.com/DKhairetdinov/SDEK-test.git`

2. Установить зависимости:

`npm install`

3. Запустить проект: 

`npm run dev`
